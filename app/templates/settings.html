如何创建 API Token

  1. 登录 Cloudflare Dashboard
  2. 进入 My Profile → API Tokens
  3. 点击 "Create Token"
  4. 选择 "Edit zone DNS" 模板
  5. 选择需要管理的域名 Zone
  6. 复制生成的 Token
